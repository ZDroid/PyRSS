# parser.py

**parser.py** is Python script for parsing RSS feeds. Requires
[feedparser](http://code.google.com/p/feedparser/) and terminal with bold
formatting support.

Licensed under the terms of MIT license.

## Usage

Run: `./parser.py $feed`

Replace `$feed` with your wanted feed.
