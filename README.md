# parser.py

**parser.py** is Python script for displaying pretty RSS feeds. It uses
[feedparser](http://code.google.com/p/feedparser/).

Licensed under the terms of MIT license.

## Usage

```bash
$ ./parser.py URL
```

`URL` can start with `http://` or `https://`.
